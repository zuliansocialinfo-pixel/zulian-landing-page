## HOW TO USE THESE PROMPTS

### Option 1: Midjourney
1. Copy prompt (after the style block)
2. Use: `/imagine prompt: [STYLE BLOCK] [SPECIFIC PROMPT]`
3. Generate at 16:10 aspect ratio for covers, 3:2 for galleries

### Option 2: DALL-E 3
1. Copy full prompt
2. Use: "Generate: [FULL PROMPT]"
3. Request specific dimensions

### Option 3: Leonardo.ai
1. Copy prompt
2. Set quality to "Ultra" or "High"
3. Use Cinematic/Professional preset
4. Request 16:10 (covers) or 3:2 (galleries)
